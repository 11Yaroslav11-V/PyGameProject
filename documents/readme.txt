Техническое задание (ТЗ).
 Создать игру "Flying duck"
1. Создать интерфейс для взаимодействия с пользователем:
   а) сделать начальное меню.
   б) сделать настройки, что бы пользователь мог ознакомиться с сутью игры.
   в) сделать выбор сложности для разнообразия игры.
   г) сделать сам интерфейс игры с движущимся задним фоном.
   д) предоставить пользователю свободное перемещение по интерфейсу.
2. Сделать нашему игроку физику и реалистичность.
3. Звуковое сопровождение:
   а) фон игры.
   б) элементы игры.
4. Сделать пасхалку с отсылкой на самую первую версию игры.
5. Записывать рекорды игрока в файл и выводить на экран.
